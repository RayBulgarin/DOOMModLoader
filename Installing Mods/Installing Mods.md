## Installing DOOMModLoader
To install DOOMModLoader, on Steam, right-click DOOM (2016) in your Steam library, and choose "*Manage*" > "*Browse local files*". (On GOG GALAXY, right-click DOOM (2016) in your game library, and choose "*Manage installation*" > "*Show folder*".) This will open a File Explorer window in your DOOM (2016) installation folder.

Download "*DOOMModLoader-Windows-x64.zip*" from [**[the Releases page]**](../releases/latest) or from one of these buttons:

<a href="../releases/latest/download/DOOMModLoader-Windows-x64.zip" title="Download for Windows"><img src="Installing Mods/Download for Windows.svg" alt="[Download for Windows]" width="186" height="32"/></a>&ensp;<a href="../releases/latest/download/DOOMModLoader-Linux-x64.zip" title="Download for Linux/SteamOS"><img src="Installing Mods/Download for Linux.svg" alt="[Download for Linux/SteamOS]" width="223" height="32"/></a>

Extract the zip into the game's installation. You should end up with "*DOOMModLoader.exe*" in the same folder as "*DOOMx64.exe*".

## Installing Mods
In your DOOM (2016) installation, find or create a folder called "*Mods*".\
Download DOOM (2016) mods from GameBanana, ModDB, Nexus Mods, or other mod-sharing sites, place the mod zips into the "*Mods*" folder (don't extract the mod zips!), and run DOOMModLoader to install them.

To uninstall mods, move the mods that you don't want out of the "*Mods*" folder, and run DOOMModLoader again.

> [!NOTE]
> 🐧 **On Linux/SteamOS,** you should right-click DOOMModLoader and choose "*Run in Konsole*", or otherwise run it in a terminal.\
> On Steam Deck, you can press Steam+X to open a virtual keyboard when you need to press Y/N to continue.

The first time that you run DOOMModLoader, you'll be asked whether to enable monthly update checks, and whether to launch the game after installing mods.

After running DOOMModLoader once, a "*DOOMModLoaderSettings.txt*" file will be created, which you can edit with a raw text editor to [**[configure settings]**](Installing-Mods-~-Settings).
